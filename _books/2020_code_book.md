关于加密和解密的通俗入门读物。不仅有古往今来关于密码术的阴谋诡计、军事秘密、学术竞争的各种故事，还能了解到加密的破绽如何可以被利用来变成破译的突破口。

和前作[《费马大定理》]({{site.baseurl}}/books/2020_fermat_last_theorem/)一样，写科普一把好手的作者 [Simon Singh](https://simonsingh.net/) 把科学史和科学知识梳理成了条理清晰的一条线索，全文阅读体验非常流畅。

每种加密方法问世时总以为固若金汤，但是聪明的密码破译者总可以敏锐地捕捉到漏洞，从而破解出明文的内容。而每当一种加密方法失效后，又总会有人想出更具技巧的加密手段，向企图破译密码的人发起新的挑战。矛和盾的较量读来非常带劲。

最后量子信息的章节读来非常令人振奋。这个潜在的 game changer，很有可能颠覆我们现有的所有的加密体系。利用量子世界的法则，不仅可以创造出锋利的矛——可以破解在经典算法下依然足够安全的 RSA 密码，甚至可以创造出无比坚实的盾——一种从理论上都无法破解的加密术。具体内容在这里就不剧透了。
