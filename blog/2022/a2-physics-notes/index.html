关于

我这份自编的 AS & A-Level Physics Notes 自2020年在我的微信公众号开源共享后，在 A-Level 圈子内已经小有名声。

在这个自建的 GitHub Page 个人站点，我也把这讲义项目作为能摆上台面的两个教学 Projects 分别放在了这里和这里。

这篇博客除了分享讲义的下载方式之外，也记录着我在创作这些资源前前后后的一些感想和絮絮叨叨。

AS 和 A2 物理电子版讲义

讲义按照 AS-Level 和 A-Level 分成了两个部分，正文的篇幅分别在150+页和220+页，排版必须是强迫症晚期的 $\LaTeX$ 纯手搓，所有的示意图均由 tikz/pgfplots 绘制的精确矢量图。讲义内容上，各种定义、描述的写法都参考了历年考题的套路，并且用不同的颜色和字体做了标识。知识点梳理的同时也编排了数量可观的例题和习题。

2022-03-19

原文：烤羚羊的 A-Level Physics Notes 更新（2022 新大纲适用）

CAIE 考试局的物理考纲（9702）在2022年有了一波更新。某些人前年开始就喊着要更新讲义，念叨了两年都没有实质性进展。果然要不是疫情把人关在家里，某些人就不会安分地把该写的讲义给敲出来。最近这周，终于是把这个坑也给填上了。讲义的下载方式见文末。

讲义正文的篇幅已经扩充到了220+页，依旧是强迫症晚期的排版——$\LaTeX$ 大法好，谁用谁知道！先前独立分享过的定义大集合、公式大集合，再经历了一轮重新校订后，也加进讲义的附录中方便大家使用。

新考纲加入了天体物理和宇宙学、以及其他零零散散的一些内容，比如 PET 成像技术、电容充放电。这些在更新后的讲义里都已经编入。某些人一如以往地无法克制地夹带了一些私货（真的只是一些吗？），某些专题的脚注篇幅甚至比正文还长。

从新考纲中删去的放大电路、通信，以及零零散散的诸如核磁共振成像、能带理论这些，在新版讲义中并没有删去，只是标记了星号，供感兴趣的同学作为拓展阅读素材（其实我就是不高兴删，老子辛辛苦苦写的，要拿去就必须连锅端走）。

这份 A-Level Physics Notes 目前只能称得上是一个草稿，任何地方都可能会出现离奇的错误，我应该会有有一搭没一搭地做做修订工作。一旦有更新，我尽力提醒自己把文件部署到上面提到的 GitHub 和百度网盘。

2020-08-31

原文：新学期开学发布：烤羚羊的 AS-Level Physics Notes

继今年年初把烂尾多年的 A2 讲义 敲完，AS 的讲义这个月也完成了初稿的审校。立的 flag 能够完成，这感觉真的是很棒的！

正逢9月1日开学日，诚意发布这份150+页篇幅、精美排版的 AS Physics 讲义。

之后计划中的更新包括：

  • 将先前独立的定义大集合、公式大集合重新校订一遍后，作为附录A和附录B加进讲义
  • 给每个章节编排一些章后习题（end-of-chapter questions）。附上习题后，这小破文档甚至可以称作是个教材了。目前这项工作已经完成近80%了，想要带习题版讲义的极客们，可以将主文件中定义的 \def\includequestions{0} 修改为 \def\includequestions{1}，现有的编好的不完全版习题就可以编译了。
  • 再添加一章实验技能（practical skills）的章节，不过目测这章会非常难产。。。

和先前 A2 讲义一样，敲完这份 AS 讲义的初衷，主要也不是为了造福学生，更多的颇有私心地只是为给自己订的目标有一个交待。在写作时，尽管主线跟着 CAIE 考试局的大纲在走，但也忍不住地夹带了很多私货、一些自己窃以为很有趣的拓展和讨论。所有读者会冷不丁地发现在部分章节中，脚注的内容会比正文还多。物理如此美妙，为什么不把我知道的那些有趣的东西写下来呢？

然而，我相信我在这份 notes 里倾注的精力最终还是可以让学习这门课程的同学有所收获。今年秋季的 A-Level 考试马上就要到来，先前夏季的常规考试被破天荒地全部取消，很多 A-Level 考生们的学业都受到的极大的影响，有不少同学卯足劲准备在10月份好好打个漂亮仗。我希望这小破讲义能真的帮助到复习迎考的同学，我也相信这份讲义对自主学习能力强的同学会很有帮助。

讲义中的错误肯定在所难免。虽然我已经比较仔细地审阅了两遍，但是应该没人更了解我自己的尿性，个中脑子犯抽的错误，阅读时千万留个心眼。另外，我大致算了算，整个工程大约 15,000 行的 $\LaTeX$ 源码，细细想想还是蛮刺激的，之后维护工作任重道远，所以很有可能我对纠错、更新不会太上心。

2020-04-01

原文：愚人节不忽悠发布：烤羚羊的 A-Level Physics Notes

利用今年春节以来死宅在家的时间，烤羚羊终于把画了N多年的饼给糊了出来。念念叨叨了N多年的 A-Level Physics Lecture Notes，在连续多个通宵化身键盘侠后，千呼万唤始出来。

就近也不挑日子，虽说正巧是愚人节，但是在此绝不忽悠，诚意发布这份200+页篇幅、精美排版的讲义。

说来这份 notes，早在2013年烤羚羊第一年教 A2 物理就已经有了雏形。随着教学经验的逐年积累，对大纲、对考试题型早已烂熟于胸，脑袋里积攒了很多素材，教案几乎已经刻在了脑子里，上课要讲的内容也已固定成了一种模板。这些东西，一直都想落实成数字化的 pdf 文档，然而出于自己对文字排版和插图的强迫症，深知工程量浩大，这份 notes 一直写写停停，在很长一段时间内距离完工似乎都遥遥无期。

也许要在此感谢下当下肆虐的疫情，让烤羚羊能够老老实实宅家，闭门造车。也许也要感谢新出生的小宝宝，每天凌晨定点要吵奶吃，可以让我假借守他这顿凌晨夜奶之名来熬夜干活。总之，当写出第一稿，做完初步的审校工作后，还是很有成就感。曾经读P神 Joseph Polchinski 写的回忆录时，他在引子中的一句话很触动我：

This (autobiography) is not written for any particular audience, but just to give myself a goal. （arXiv:1708.09093）

想来我敲完这份 notes，主要目的似乎也不是为了造福学生，更多的似乎是为了给自己订的目标有一个交待。出于这个目的，我在写作时，尽管主线跟着 CAIE 考试局的大纲在走，但也忍不住地夹带了一些私货、一些自己窃以为跟大纲内容比较贴合、并且很有趣的拓展和讨论。物理理论如此美妙，为什么不把我知道的那些有趣的东西写下来呢？

然而，我相信我在这份 notes 里倾注的精力最终还是可以让学习这门课程的同学有所收获。尤其又是在如此特殊的一个时期，A-Level 考试破天荒地全部取消，应届毕业生们尚且不知何去何从，所有的 A-Level 考生们的学业都受到的极大的影响。我希望我也能尽一点点绵薄之力，帮助有需要的同学不要耽误学业。

最后扯扯后面的计划。

这份 A2 Physics Notes 目前仅是第一稿完成，指不定哪里会冒出一堆打印错误，我应该会有一阵没一阵地做做修订工作。

CAIE 考试局在2022年后会对现有的大纲做一次调整。放大电路（amplifier circuits/electronics）和通信（telecommunication）这两章被扫地出门，我倒是挺欣喜地看到 CAIE 终于弃暗投明地把天体物理（Astrophysics）和宇宙学（Cosmology）作为新内容加了进来。这两大块内容在其他考试局的 A-Level 物理大纲中都有涉及，我也有过几轮的教学经验。我近期会琢磨下 CAIE 新大纲的精神，试着把这两章的内容增补到现有的 notes 中去。

我最近也有打算把我同样烂尾很久很久的 AS Physics Notes 拿出来回炉重造，这个工程量也是细思极恐，所以等完整的第一稿出来，也许会要比较久，希望我能 push 自己让进度条拉得快一些吧。
