解码者 The Code Breaker

Walter Isaacson  ·  2022

2024-09-02  · 

biology   popular-science   biography   Nobel

biology   popular-science   biography


Walter Isaacson 不愧写传记的一把好手。近十来年在各大排行榜上均有相当热度的 Steve Jobs，Albert Einstein，Elon Musk 的传记，皆出自他手。这本较新的 Jennifer Doudna 传记，叙事结构完整，逻辑清晰，专业内容的深入程度远超我的预期。中译本的翻译也非常棒。

推荐指数：9.0/10

通读全书前，我以为这本书单纯是 UC Berkeley 生物化学大佬 Jennifer Doudna 的个人传记。Doudna 将幼时对自然的兴趣转换成毕生事业的故事本身固然很精彩：生涯早期，她通过揭示核酶的结构和其生物学功能的基本机制开始展露头角。在 Berkeley 拥有独立实验室后，她在 CRISPR-Cas9 基因编辑技术的领域作出开创性贡献，也因此获得了2020年的诺贝尔化学奖。随着 CRISPR-Cas9 技术快速发展，她又致力于将此技术应用于遗传疾病治疗、病毒检测、疫苗开发等人类健康议题的研究，担任多个项目的掌舵人。相信 Doudna 乘风破浪的成长历程，可以鼓舞到有志投身于科研的年轻人，尤其是未来的女性科学家。

然而这本书涉及的话题远不止 Jennifer Doudna 一条主线。正如副标题暗示的，这本书也是一部科学家认识基因、逐渐掌握基因编辑技术的科学史。科学发现的旅程总是漫长又充满波折，CRISPR-Cas9 的技术突破来自于许多科学家的共同努力：最先认识到 CRISPR 可能跟细菌免疫系统有关的的 Mojica，通过酸奶嗜热链球菌实验验证 CRISPR 确实是一种适应性免疫系统的 Horvath 和 Barrangouz，确定 CRISPR 系统的靶向分子就是 DNA 的 Martini 和 Sontheimer，揭示引导 Cas9 蛋白至靶标的核心机制并创造出简化版的单一合成导向 RNA 的 Doudna 和 Charpentier 团队，以及首先将 CRISPR-Cas9 技术成功用于真核细胞基因编辑的张锋团队……这些英雄谱中的每个人，都站在了前人的肩膀上，但只有当他们的发现串到一起，拼图的全貌才渐渐浮现。

当然，全世界的研究团队之间除了倾力合作，也充满竞争，乃至关于优先权的争执。全书相当篇幅介绍了 Doudna 和张锋团队之间的专利之争、哪方的工作更为重要之争，读来令人颇感唏嘘。

对于基因编辑技术这把开启潘多拉魔盒的钥匙，作者也对可能产生的伦理、政治哲学问题展开了讨论，其中躲不开的一个话题自然是2018年贺建奎的基因编辑婴儿事件。当技术进步的步伐远快于社会规范的跟进时，如何在鼓励技术突破和约束道德准则中间找到平衡点，可能比基因编辑技术本身还要复杂。
